# MBT Streak

SwiftUI iPhone-only app for sports streaks, team form, matchup momentum and live trend awareness.

Slogan: See the streak. Know the momentum.

## Features

- Streak cards for hot, scoring, unbeaten, defensive and slump trends.
- Team form dashboard with compact recent results and momentum scores.
- Two-team comparison with signal board and matchup edge.
- Live center with sport-aware fixtures, live/final sections, clock/status labels and streak impact notes.
- Personal streak alerts with a creation form, presets, custom rules and toggles.
- Favorite teams with mini logos and quick star controls.
- Smart recap after a round.

## Build

Open `MBT Streak.xcodeproj` in Xcode or run:

```sh
xcodebuild -project "MBT Streak.xcodeproj" -scheme "MBT Streak" -sdk iphonesimulator -destination "generic/platform=iOS Simulator" build
```
